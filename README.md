ClearMind — AI Wellness & Productivity Assistant
Your mind, clearer every day.
ClearMind is an AI-powered assistant that helps users manage stress, improve focus, and plan their day through a simple conversational interface.

✨ Features
Emotion-aware responses
Smart daily planning
Wellness & stress support
Interactive chat experience

Inspiration
Many people struggle with stress and lack of focus, yet most tools ignore emotional state. ClearMind was built to understand how users feel before helping them act.

How It Works
Users share how they feel → ClearMind analyzes input → Provides personalized guidance.

Built With
HTML, CSS, JavaScript
Custom chat UI
Rule-based AI (keyword detection)
